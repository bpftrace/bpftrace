Demonstrations of killsnoop, the Linux bpftrace/eBPF version.



This traces signals sent via the kill(),tkill(),tgkill() syscall. For example:

# ./killsnoop.bt
Attaching 7 probes...
Tracing kill(),tkill(),tgkill() signals... Hit Ctrl-C to end.
TIME                 PID COMM              SIG     TPID RESULT
17:26:10.561758   648047 containerd         23   648055      0
17:26:11.875708  1504730 bash                2  1504730      0
17:26:12.046966  1514031 crio               23  1514035      0

The first line showed a SIGURG (23) sent from PID 648047 (containerd) to
PID 648055. The result, 0, means success. The next line shows the same signal
sent, which resulted in -3, a failure (likely because the target process
no longer existed).


There is another version of this tool in bcc: https://github.com/iovisor/bcc
The bcc version provides command line options to customize the output.
